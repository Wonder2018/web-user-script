# 帖子筛选工具

在帖子列表头部添加了几个按钮支持以下功能：

1. 过滤被标记为已解决的帖子
2. 过滤回复数较高的帖子
3. 过滤悬赏 CB 较少的帖子
4. 还原被过滤的帖子
5. 使用正则表达式过滤帖子
6. 支持设置按钮添加位置
7. 支持设置各功能开启状态
